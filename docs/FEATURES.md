# Feature Backlog

Mirrors the tiered model validated against the competitor (mywedsite.gr), plus our AI differentiator. Check items off as built.

## Tier 1 — Kite (€150) — Core
- [ ] Cover photo upload
- [ ] Up to 5 gallery/slider photos
- [ ] Couple names, date, time, venue text
- [ ] Custom invitation message text
- [ ] Accent color picker (curated palette, not a free picker)
- [ ] Font picker (3–5 curated pairings)
- [ ] RSVP form: attending y/n, adult/child count, dietary notes
- [ ] Shareable public link + QR code
- [ ] Guest response list (dashboard table, CSV export)
- [ ] Mobile-responsive public invitation page
- [ ] Optional entry password/PIN

## Tier 2 — Balloon (€260) — Adds
- [ ] Up to 10 gallery photos
- [ ] Seating chart builder (tables, seat counts, group names)
- [ ] Guest quiz (multiple choice, results emailed to host)
- [ ] Countdown timer to ceremony
- [ ] Interactive venue map (Google Maps embed, ceremony + reception pins)
- [ ] Visit analytics (views over time)
- [ ] Live guest chat / guestbook wall
- [ ] Add-to-calendar (Apple/Google/Outlook/Yahoo)

## Tier 3 — Rocket (€370) — Adds
- [ ] Unlimited photos
- [ ] Seating chart Pro: RSVP auto-syncs into table assignments; guests scan a QR at the venue to find their seat
- [ ] Gift registry / IBAN-IRIS display with copy-to-clipboard
- [ ] Hotel recommendations list (name, price, map link, notes)
- [ ] Background music (opt-in, starts muted)
- [ ] QR photo gallery — guests upload photos/videos during/after the event, host can download all
- [ ] Second language toggle
- [ ] PDF export of the final invitation (keepsake)

## Cross-Tier / Platform Features
- [ ] Editor autosave + explicit "Publish" step
- [ ] Draft vs Published vs Archived invitation states
- [ ] Dashboard: guest CRUD (manual add for phone/in-person RSVPs), CSV import
- [ ] Email notifications to host on each new RSVP (and to guest as confirmation)
- [ ] Deposit-based checkout (50% now / 50% on delivery) — see INFRASTRUCTURE.md
- [ ] Account settings (profile, password reset, delete account/data)

## Our Differentiator — AI Photo Studio
- [ ] Upload a photo → describe a mood/edit in plain language → get a restyled version back
- [ ] Style presets as one-click shortcuts (Golden Hour, Romantic Soft, Editorial B&W, Vibrant Garden, plus free-text)
- [ ] Multi-turn refinement ("now make it warmer", "remove the trash can in the background")
- [ ] Before/after compare slider
- See **AI_IMAGE_EDITING.md** for the technical spec.
